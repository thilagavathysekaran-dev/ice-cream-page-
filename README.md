This project is a simple Ice Cream website created using HTML and CSS.
The page showcases the ice cream menu, allows users to give feedback, and includes a registration box for customers.

Features:
Ice cream menu display
Customer feedback section
Registration box for user details
Simple and user-friendly layout

Technologies Used:
HTML5
CSS(flex)

Files Included:
index.html – Main webpage structure
style.css – Styling and layout design
